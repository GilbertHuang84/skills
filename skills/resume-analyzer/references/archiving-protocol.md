# 简历审阅存档协议

> 本文件定义 resume-analyzer 审阅报告的存档格式和模板。

---

## 存档目录结构

```
data/
└── {用户标识}/
    ├── internal-resume.md              # 内部简历（个人档案馆）
    ├── profile.md                      # 用户画像
    └── resume-reviews/
        ├── review-2026-04-20.md        # 按日期存储
        ├── review-2026-04-20-2.md      # 同一天第二次审阅
        └── review-2026-05-15.md
```

---

## 回写标记规范

当 resume-analyzer 向 internal-resume.md 或 profile.md 回写内容时，使用以下标记格式：

```markdown
<!-- resume-analyzer review YYYY-MM-DD 回写开始 -->
回写的内容...
<!-- resume-analyzer review YYYY-MM-DD 回写结束 -->
```

**回写原则：**
- 只追加，不删除已有内容
- 每次回写使用独立的标记块
- 标记中包含审阅日期，便于追溯
