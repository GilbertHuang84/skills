# 国家战略与政策分析框架

> 本文件提供政策分析的**方法论和评估框架**，不包含具体政策数据。
> 具体政策内容应在生成报告时通过 WebSearch 实时搜索获取，确保信息时效性。

---

## 一、概述：为什么国家政策分析对职业决策至关重要

### 1.1 政策是产业发展的底层逻辑

在中国市场环境下，国家政策对产业发展的影响远超一般市场经济体。政策不仅决定：

- **行业增长曲线**：政策支持的行业往往获得数倍于市场平均增速的发展空间
- **资金流向**：政府引导基金、税收优惠、专项补贴直接影响企业融资能力和扩张意愿
- **人才需求**：政策驱动的产业升级会催生全新岗位类型，同时淘汰落后技能
- **企业生存**：政策收紧可能导致整个细分赛道在短期内急剧收缩

### 1.2 政策分析的局限性

- 政策执行存在地区差异，中央政策与地方落地可能有时间差
- 政策方向可能随国际形势变化而调整
- 政策利好不等于个人能力匹配，仍需结合个人竞争力评估
- 政策分析是辅助决策工具，不能替代对行业基本面和企业个体情况的判断

## 二、政策分析执行流程

> **核心原则：所有政策数据必须通过实时搜索获取，不得使用本文件中的任何固定数据。**

### 2.1 Step 1: 从简历推导相关政策领域

根据候选人的工作经历，映射到可能相关的政策领域：

| 简历信号 | 可能相关的政策领域 |
|---------|------------------|
| 某新兴技术相关经历 | 该技术的产业政策、安全监管政策 |
| 某内容/创意产业经历 | 文化数字化战略、数字内容产业政策 |
| 某虚拟/沉浸式技术经历 | 文化数字化战略、相关产业政策 |
| 数据平台/数据治理经历 | 数据要素市场、数字中国建设 |
| 基础设施/云平台经历 | 数字中国建设、新质生产力 |
| 某传统行业经历 | 该行业的数字化转型政策、新质生产力 |

**只分析有简历信号支撑的政策领域，不列举无关领域。**

### 2.2 Step 2: 实时搜索政策信息

对每个相关政策领域，搜索以下内容：

**搜索关键词模板**：
- `"{政策领域名} 2025 2026 最新政策"`
- `"{政策领域名} 产业发展现状 趋势"`
- `"{目标公司所在行业} 政策影响"`

**搜索预算**：1-3次搜索（将多个领域合并为一次搜索）。

**需要获取的信息**：
1. 最新政策文件名称和发布时间
2. 政策的核心方向和重点支持领域
3. 对行业/岗位的具体影响
4. 政策所处的阶段（出台初期/执行期/调整期）

### 2.3 Step 3: 评估政策对具体公司的影响

对报告中涉及的每家公司，评估：

| 评估项 | 具体问题 | 判断方法 |
|--------|---------|---------|
| **主营业务政策对齐** | 公司核心业务是否在政策鼓励的方向上？ | 对照搜索到的政策内容匹配公司业务 |
| **政策红利享受度** | 公司是否能直接享受政策红利？ | 查看公司是否在政策试点名单、是否获得政府项目 |
| **政策风险暴露度** | 公司业务是否面临政策收紧的风险？ | 评估监管变化对公司收入的潜在影响 |
| **政策驱动增长潜力** | 政策变化是否带来新的增长机会？ | 评估政策新方向与公司能力的匹配度 |

**公司政策评级输出**：S / A / B / C / D

### 2.4 Step 4: 评估政策对具体岗位的影响

对每个目标岗位，评估：

| 评估项 | 具体问题 |
|--------|---------|
| **技能政策溢价** | 该岗位所需技能是否因政策而获得薪资溢价？ |
| **需求增长趋势** | 政策是否推动该岗位的市场需求增长？ |
| **替代风险评估** | 政策推动的技术变革是否威胁该岗位？ |
| **3-5年需求预测** | 政策如何影响该岗位的中长期需求？ |

## 三、政策领域阶段评估框架

> 对搜索到的每个政策领域，使用以下框架进行结构化评估。

### 3.1 阶段判断标准

| 阶段 | 特征 | 入局建议 |
|------|------|---------|
| 📈 快速上升期 | 政策密集出台、资本涌入、岗位需求爆发 | 好时机，越早入局越好 |
| 📈 上升期 | 政策方向明确、基础设施建设中 | 较好时机，建议1年内行动 |
| 📈 政策驱动上升期 | 概念先行、落地参差不齐 | 选择性入局，找具体落地场景 |
| 📊 基础建设期 | 制度框架建立、基础设施搭建中 | 提前布局，适合长期视角 |
| ➡️ 平稳恢复期 | 经过调整后恢复增长 | 稳健入局，风险可控 |
| ➡️ 调整转型期 | 技术变革导致行业重构 | 转型入局，适合有传统经验+新技能的人 |
| 📉 概念降温期 | 热度消退、回归理性 | 低成本积累期，适合长期看好的人 |

### 3.2 评估输出格式

对每个政策领域，输出：

| 字段 | 内容 |
|------|------|
| 政策领域 | 从简历推导出的领域名称 |
| 简历契合度 | ★★★★★ ~ ★☆☆☆☆（基于简历经历与该领域的关联度） |
| 当前阶段 | 从搜索结果判断（上升期/建设期/转型期/降温期等） |
| 风向 | 顺风/中性/逆风 |
| 基建状态 | 从搜索结果判断 |
| 入局建议 | 基于以上分析的综合建议 |

## 四、报告中的政策分析输出规范

### 4.1 政策环境概述表

在报告"公司分析"部分的政策概述中，使用以下表格格式：

```
| 政策领域 | 简历契合度 | 风向 | 阶段 | 基建状态 | 入局建议 |
```

**数据来源**：全部来自实时搜索结果，不使用固定数据。

### 4.2 公司政策对齐评估

在每家公司的分析中，包含：

```
政策契合度评级：S/A/B/C/D

核心业务与政策对应：
| 公司业务方向 | 对应政策领域 | 契合度 | 说明 |
|-------------|-------------|--------|------|

发展前景评估：
- 行业前景：[上升期/平稳期/调整期/衰退期]
- 政策红利：[直接受益/间接受益/无明显影响/面临风险]
- 增长潜力：[高/中/低]
- 主要政策风险：[描述]
```

### 4.3 岗位政策契合评估

在每个岗位的分析中，包含：

```
政策契合度评级：S/A/B/C/D

| 影响维度 | 评估 | 说明 |
|---------|------|------|
| 技能溢价 | 有/无 | [具体说明] |
| 需求趋势 | 上升/平稳/下降 | [具体说明] |
| 替代风险 | 高/中/低 | [具体说明] |
| 新兴机会 | 有/无 | [具体说明] |

未来3-5年需求预测：
- 1年内：[预测]
- 1-3年：[预测]
- 3-5年：[预测]
```

## 五、搜索来源参考

实时搜索时，优先使用以下来源：

| 来源类型 | 具体来源 | 用途 |
|---------|---------|------|
| 政府官网 | 国务院、工信部、文旅部、广电总局 | 获取原始政策文件 |
| 行业协会 | 相关行业协会官网 | 获取行业解读和执行动态 |
| 研究平台 | 36氪、虎嗅、钛媒体 | 获取产业分析和投融资动态 |
| 政策数据库 | 北大法宝、威科先行 | 获取政策全文和修订历史 |

## 六、参考来源格式

报告中引用的政策来源，按 GB/T 7714-2015 电子资源格式著录：

> 主要责任者. 题名[EB/OL]. (更新日期)[引用日期]. URL.

示例：
> 国务院. 新一代人工智能发展规划: 国发〔2017〕35号[EB/OL]. (2017-07-20)[2026-04-18]. https://www.gov.cn/zhengce/content/2017-07/20/content_5211996.htm
